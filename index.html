ABOUT US

Team Uranium, led by Grace, Richard, and Jack, is an organization founded by their shared love of Pokemon. Their main goal is to bring fun and competition into the modern world with their new Pokemon memory game, PokeFlip.

ABOUT THE GAME

This app was created for Alchemy Code Lab 301 final project. It is a memory game. MUCH FUN. It utilizes PokeApi.co to obtain information and images on pokemon. A server was created to route requests for PokeApi, as well as GET/POST requests for scores.
